This folder conains the files related to ica08. 

"ica08_borrowedSollution" have a full functional code, that uses an GUI filled with ascii characters to simulate the River Crossing game. "ica08_statesPictures" is a colluction of all the pictures that are related to our own functional River Crossing game.

Main folder contains states files (PNG image files), with corresponding numbers. The numbers indicates the order of the pictures within the game. The "ica08_rivercrossingGraphic.py" is the main file to use when running the game.
